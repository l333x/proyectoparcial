TechVision Analytics es una plataforma web desarrollada como parte del primer parcial de la carrera Negocios Digitales de la Universidad Politécnica Salesiana. El proyecto tiene como objetivo diseñar una herramienta moderna, intuitiva y responsive que permita visualizar información empresarial en tiempo real mediante dashboards interactivos y módulos de análisis.

La solución integra diferentes componentes clave: una página inicial informativa con funciones de inteligencia artificial, un dashboard dinámico para el seguimiento de KPIs, un formulario de contacto validado y funcional, y una estructura modular que facilita la escalabilidad del sistema. Su desarrollo se realizó aplicando buenas prácticas de programación, diseño responsivo, consumo de datos simulados y uso de librerías especializadas para visualización gráfica.

Este proyecto busca demostrar el uso de tecnologías web actuales aplicadas a la toma de decisiones, la gestión operativa y la experiencia del usuario dentro del entorno digital empresarial.
